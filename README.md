# 📄 Life Sciences Research Article Insight Extractor

A GenAI-powered application that analyzes Life Sciences research papers using the Gemini API.

### Features

* Upload research paper PDFs
* Generate structured summaries
* Ask questions based on paper content
* Streamlit-based interactive UI

### Tech Stack

Python • Streamlit • Gemini API • PyMuPDF

### Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
